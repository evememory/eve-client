# Eve Memory

Eve is your durable memory layer for relevant context, preferences, and prior decisions.

## Use Eve When

- Prior context or preferences would improve the next answer.
- A user gives a stable preference, correction, or decision.
- You want to preserve a useful outcome for a future session.

## Read Discipline

- Search Eve before saying prior context is missing.
- Prefer relevant existing memories over repeating discovery work.

## Write Discipline

- Store concise durable memories, not full transcripts.
- Keep entries specific enough to be useful later.
- Do not store secrets, API keys, bearer tokens, passwords, or private credentials.

MCP endpoint: `https://mcp.evemem.com/mcp`
